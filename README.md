# fire-vue-store
Vue.js 를 통한 프론트와 firebase를 사용한 백엔드로 웹앱 개발하기 전 기초 다지기


## Project setup
```
npm install
```

### Compiles and hot-reloads for development

npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files

npm run lint
```
